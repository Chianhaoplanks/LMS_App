# How to Use This App

1. **Fork** or **clone** this repository to your local machine.

2. **Open two terminal windows or tabs**:
   - One for the frontend (`client` directory)
   - One for the backend (`server` directory)

3. In each directory, run:

   ```bash
   npm install

4. Start the app:
   - Frontend:
   
        ```bash
        npm start
    - Backend:
        ```bash
        node index.js
